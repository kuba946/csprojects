Do uruchomienia aplikacji jest potrzebny komputer, który swoimi parametrami technicznymi umożliwia zainstalowanie systemu operacyjnego Windows 10 lub nowszego. 

Następnie należy pobrać i zainstalować narzędzie SQL Server Express 2019 ze strony producenta https://www.microsoft.com/pl-pl/sql-server/sql-server-downloads.
 
Kolejnym krokiem jest zainstalowanie środowiska deweloperskiego Visual Studio 2022 ze strony https://visualstudio.microsoft.com/pl/vs/community/.
 
Następnie należy uruchomić projekt aplikacji i we wbudowanej konsoli menedżera pakietów NuGet wykonać komendę „Update-Database”.
 

Po wykonaniu powyższych kroków, aplikacja jest gotowa do uruchomienia. Po pierwszym uruchomieniu w aplikacji jest zarejestrowany użytkownik roli Administratora o loginie admin@admin.com, oraz haśle „P@ssw0rd”.
